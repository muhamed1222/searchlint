# SearchLint FAQ

Status date: 2026-06-30

This FAQ is public website source material. It does not approve SearchLint 1.0
release or claim that final packages, cloud services, or marketplace listings
are live.

## What is SearchLint?

SearchLint is a deterministic SEO diagnostics product for local development, CI,
editor diagnostics, crawling, reporting, and future hosted workflows. Rule
execution does not require AI.

## Does SearchLint add code to production sites?

No. The SearchLint 1.0 invariant is zero production JavaScript, zero production
HTTP requests, and zero production background processes or hooks on the end
user's production site.

## Are npm packages and the VS Code extension published?

Do not claim final npm or VS Code Marketplace publication until the publication
release gates pass and the final release gate is approved.

## Does SearchLint support Google and Yandex?

The product scope includes Google and Yandex evidence sources. Public claims
about live provider integrations require dated provider acceptance evidence.

## Is the cloud platform live?

The full 1.0 scope includes dashboard, API, workers, hosted reports, auth,
billing, provider integrations, observability, and agency workflows. Live cloud
availability remains release-gated until deployment and owner evidence pass.

## Where should security issues be reported?

Public support and vulnerability disclosure routing must use the approved
support and security policy documents before launch.
