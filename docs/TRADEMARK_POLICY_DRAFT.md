# SearchLint Trademark Policy Draft

Status date: 2026-06-30

This draft defines the trademark boundary that must be reviewed before any
public SearchLint release. It is not legal approval.

## Reserved Marks

The SearchLint name, logo, package names, service names, and related product
marks are reserved by the project owner. The Apache-2.0 license grants copyright
and patent rights for the covered public code, but it does not grant trademark
rights.

## Permitted Descriptive Use

Users may refer to SearchLint to describe compatibility, installation, command
usage, bug reports, tutorials, or factual statements about the project, provided
the use is not misleading and does not imply sponsorship or endorsement.

## Restricted Use

Do not use SearchLint marks for a competing product, hosted service, domain,
package, extension, logo, paid offering, or marketing campaign without written
approval from the project owner.

## Release Gate

Before public release, qualified legal review must approve this draft together
with `LICENSE`, `NOTICE`, package metadata, VS Code Marketplace copy, public
website copy, and the public/private repository boundary.
