# SearchLint Demo Project

Status date: 2026-06-30

This demo project is public website source material for onboarding examples. It
shows the shape of a `searchlint.seo` route contract without claiming that
SearchLint 1.0 packages have been published.

## Files

- `searchlint.seo` shows a minimal site contract with product and admin routes.

## Usage

Use this fixture to explain local CLI onboarding, route coverage, and rule
configuration. The demo is documentation evidence, not live production evidence.
